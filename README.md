# Ender3_upgrade
# Ender-3 Marlin Upgrade Project

Custom Marlin firmware setup for Ender-3 with BIGTREETECH SKR MINI E3 V3.0 and TFT35-E3 V3.0.1.

---

##  Hardware
- Ender-3 V1
- BIGTREETECH SKR MINI E3 V3.0
- TFT35-E3 V3.0.1
- Auto Bed Leveling Sensor (CR-Touch / BLTouch)

---

##  Firmware
- Marlin 2.x
- Auto Bed Level Enabled
- Thermal Protection Enabled
- Custom Fan Control

---

##  Folder Structure
- `marlin_firmware/` → Firmware + Config
- `wiring/` → Wiring diagram
- `images/` → Printer setup
- `3d_mods/` → STL upgrades

---
